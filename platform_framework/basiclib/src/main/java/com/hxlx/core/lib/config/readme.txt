相应配置:

AppConfig                     ：配置应用使用的渠道号，和设备号
CrashReportConfig             ：配置crash报告
DBConfig                      ：配置数据库
FileConfig                    ：配置文件路径及初始化外部文件路径
HttpConfig                    ：配置网络请求初始化网络请求参数
IntentParameter               ：配置Intent传参的key等公共常量
NetWorkConfig                 ：配置网络实时监听回调系统版本5.0以后使用
RestApiPath                   ：配置接口请求路径
SharedPreferencesKeys         ：配置SharedPreferences存储的key
